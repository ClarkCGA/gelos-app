Geospatial Exploration of Latent Observation Space (GELOS)

The Geospatial Exploration of Latent Observation Space (GELOS) web app was developed by the Center for Geospatial Analytics at Clark University. It visualizes t-SNE embeddings generated from geospatial image chips using a foundation model.

  • Choose Globe or Map mode from the button in the lower-left corner. Use the Models menu to select a geospatial foundation model. The Images menu toggles between Sentinel-1, Sentinel-2, and Landsat imagery thumbnails.
  • Click a point on either the map or the plot to highlight the corresponding point in both views. You can select multiple points using the plot’s Box Select or Lasso Select tools, or by drawing polygons directly on the map.
  • Hover over a point to view its attributes, including ID, t-SNE values, and location. Point colors represent different land-cover classes.
  • Use the checkboxes in the legend to filter and explore specific land-cover classes.
